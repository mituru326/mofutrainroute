# もふくらふと 乗換案内サイト

このリポジトリは、**Minecraftサーバー「MOFUCRAFT!!」**内の路線ネットワークにおける**乗換案内・経路検索**を目的としたWebアプリケーションのソースコードです。

---

## ソフトウェアの詳細説明

### 1. 機能

* **経路検索:** 指定した**出発駅**と**到着駅**から、対応路線内での最短乗換経路を探索します。

### 2. 対応路線

本プログラムにて現時点(2025/11/20)で標準で検索できる鉄道会社および路線網は以下の通りです

| 会社名 | 路線名 | 区間（備考） |
| :--- | :--- | :--- |
| **鯖鉄** | 南北線(南/北), 東西線(東/西), 白馬線, 舞雀線 | 全線 |
| **南陽電鉄** | 南陽本線, 白馬線, 笹山線, 和泉線 | 全線 |
| **極楽園鉄道** | 極雪本線 | 全線 |
| **稲急** | 稲急本線, 白馬線 | 全線 |
| **和泉市交通局** | 和泉市電 | 全線 |

### 3. 内容

* **HTML (`index.html`):** Web
* **CSS (`style.css`):** レイアウト
* **JavaScript (`script.js`):** 経路検索計算

**注記:** 経路探索アルゴリズム（**ダイクストラ法**etc）は、`script.js`内に記述されています
